# VSCO Canvas 产品分析

## 1. 拼贴能力（基础能力）

### 支持功能

- 添加图片（本地或 VSCO 图库）
- 自由拖拽
- 缩放旋转
- 图层叠加
- 透明度调整
- 裁剪

### 核心特点

- **无网格限制**：不同于模板拼图，强调自由感
- **手工质感**：营造手工拼贴的感觉

---

## 2. 文本表达能力

### 支持功能

- 添加文字
- 调整字体、大小、对齐

### 产品定位

- 文本作为**情绪补充**，而非主要设计元素
- 不追求 Canva 式的复杂排版能力

---

## 3. 情绪表达结构（核心优势）

VSCO Canvas 的关键不是功能堆砌，而是**氛围构建能力**：

| 维度       | 说明                 |
| ---------- | -------------------- |
| 颜色呼应   | 色彩之间的协调搭配   |
| 光影情绪   | 通过光影传递情感     |
| 空间留白   | 适当的空白营造呼吸感 |
| 非对称排布 | 打破规整，追求自然   |
| 视觉流动感 | 引导视线自然游走     |

### 适用场景

- 旅行灵感板
- 品牌视觉方向
- 摄影风格测试
- 穿搭 Moodboard
- 情绪日记

### 核心定位

> 不追求规整，而追求**"感觉"**。

---

## 4. 社区传播能力

### 能力支持

- 发布到 VSCO 社区
- 被收藏、被浏览

### 特点

- 没有点赞压力
- 没有强互动机制
- 没有算法推流

### 产品理念

> "被看见的作品"，而不是"博流量的内容"。

---

## 总结

VSCO Canvas 的核心价值在于：

1. **轻量拼贴** - 自由无约束的图片组合
2. **情绪优先** - 氛围感大于功能性
3. **安静展示** - 作品导向，而非流量导向
