# Luminar Neo 功能概览

## 一、基础图像编辑功能

### 1.1 曝光与影调控制（Develop/DevelopRAW）

| 功能                       | 说明              |
| -------------------------- | ----------------- |
| Exposure（曝光）           | 整体亮度调整      |
| Contrast（对比度）         | 明暗对比控制      |
| Highlights（高光）         | 高光区域细节恢复  |
| Shadows（阴影）            | 阴影区域提亮      |
| Whites（白色色阶）         | 白场控制          |
| Blacks（黑色色阶）         | 黑场控制          |
| Smart Contrast（智能对比） | AI 驱动的对比优化 |

> 功能覆盖 Lightroom Basic 面板的全部核心曝光控制

### 1.2 白平衡与色温控制

- **Temperature（色温）** - 冷暖色调调整
- **Tint（色调）** - 绿-洋红轴调整
- **自动白平衡** - 一键自动校正
- **RAW 原始白平衡读取** - 保留相机原始设置

### 1.3 全局色彩控制

- **Saturation（饱和度）** - 整体色彩浓度
- **Vibrance（自然饱和度）** - 智能保护肤色的饱和度调整
- **Color Enhance（色彩增强）** - AI 色彩优化
- **Remove Color Cast（去色偏）** - 自动消除色偏

---

## 二、色调曲线（Tone Curve）

- RGB 曲线
- 独立通道曲线（R / G / B）
- 点曲线模式
- 曲线整体对比控制
- 高光 / 中间调 / 阴影独立调节

> 具备专业曲线功能，但操作界面比 Lightroom 简洁

---

## 三、HSL / 单色通道调整

| 调整维度             | 说明         |
| -------------------- | ------------ |
| Hue（色相）          | 改变颜色属性 |
| Saturation（饱和度） | 控制颜色浓度 |
| Luminance（明度）    | 调整颜色亮度 |

**支持颜色通道**：红、橙、黄、绿、青、蓝、紫、品红

> 基本覆盖 Lightroom 的 HSL 功能


## 总结

### 优势

作为"基础图像编辑器"，Luminar Neo 具备：

- 完整的曝光控制体系
- 完整的白平衡系统
- HSL 与曲线调整
- 锐化与降噪
- 透视与镜头校正
- 局部调整（蒙版 + AI）
- 图层系统
- 批量处理能力
- RAW 开发能力

**功能覆盖约 90% Lightroom 的基础编辑需求**

### 劣势

- 颜色管理和专业调色深度略弱于 Lightroom
- 批量管理能力弱于 Lightroom
- Catalog 专业度低于 Lightroom
- 更偏向 AI 辅助而非完全手动精调
- 生成式 AI 功能需要联网（云端处理）
