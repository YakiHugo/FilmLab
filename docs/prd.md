# FilmLab PRD（移动端优先 / 新手傻瓜式 / 工作台 IA）

## 1. 产品定位
FilmLab 是一个移动端优先、本地优先的摄影后期入门工具。面向初学者，以“导入 → 选风格 → 导出”的三步主线实现傻瓜式出片，同时保留素材库与批量应用能力，确保中途退出可恢复、批量一致性可控。

## 2. 目标用户
- 初学者：不理解参数也能快速出片；希望结果可预期、学习成本最低。

## 3. 体验原则（乔布斯式设计逻辑）
- 默认即最佳：第一屏就能得到满意效果
- 单一主行动：每一步只保留一个主按钮
- 渐进披露：进阶能力不打断主线
- 预览优先：画面始终是视觉中心
- 低噪清晰：减少控件与术语暴露

## 4. 核心流程（移动端单线）
导入素材 → 自动分组/推荐 → 选择风格包 → 一键导出
（可选）微调与进阶预设在二级面板打开。

## 5. 信息架构（单工作台）
- 单一工作台 Workspace，通过“步骤条 + 状态”切换
- 三个步骤：素材 / 风格 / 导出
- 进阶入口：风格步骤内的“高级”与单张微调

## 6. 布局策略（响应式）
- 移动端：素材库为底部抽屉，主区为预览 + 当前步骤面板，主按钮固定底部
- 平板/桌面：素材库为左侧栏（导入后固定），预览在中间，控制面板在右侧

## 7. MVP 功能范围
- 素材导入与管理（JPG/PNG、多选/拖拽）
- 素材库（导入后左侧区域；移动端底部抽屉）
- 自动分组/选择集批处理（风格包批量应用）
- 风格包与预设滤镜（新手卡片式选择 + 进阶自定义保存）
- 单张微调（仅 3 个主控，其他为高级折叠）
- 导出（格式/质量/尺寸配置）
- 本地持久化（IndexedDB）

## 8. 非目标（当前不做）
- 云端协作/同步
- RAW 全流程
- 完整离屏渲染与导出队列

## 10. 风险与注意
