# FilmLab PRD（当前范围摘要）

## 1. 产品定位
FilmLab 是一个本地优先的摄影后期工作流 Demo，目标是用“批量风格统一 + 单张精修”的方式提升出片效率，并为后续 AI 训练/推荐能力预留接口。

## 2. 目标用户
- 轻量摄影爱好者：希望快速统一风格、获得胶片感效果。
- 活动/旅拍场景：需要批量处理与一致性输出。
- 初学者：希望看到可解释的调色建议（当前为占位）。

## 3. 核心流程
导入素材 → 选择/分组 → 批量套用预设 → 单张微调 → 导出

## 4. MVP 功能范围
- 资产导入与管理（JPG/PNG、多选/拖拽）
- 分组/选择集批处理（应用 preset 与强度）
- 单张编辑（基础调整项、原图对比）
- 导出（当前为原图导出占位）
- 本地持久化（IndexedDB）

## 5. 非目标（当前不做）
- 云端协作/同步
- RAW 全流程
- AI 训练与真实推荐（目前仅占位）
- 完整离屏渲染与导出队列

- [x] 资产导入与本地存储（IndexedDB）
