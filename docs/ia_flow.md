# FilmLab 信息架构与页面结构（移动端优先）

## 1. 目标与原则
- 单线流程：导入 → 选风格 → 导出
- 低负担：默认推荐风格与一键应用
- 渐进披露：进阶预设与参数在二级面板
- 不中断：中途退出可恢复，素材库始终可达
- 结果优先：始终看到对比与预览

## 2. IA（单工作台 + 状态驱动）
- 单一工作台（Workspace），通过“步骤条 + 状态”切换
- 三个步骤：
  1) 素材
  2) 风格
  3) 导出
- 进阶入口：
  - 风格内的“高级”
  - 单张微调（不占主流程步骤）

## 3. 响应式布局策略
- 移动端（优先）：
  - 素材库为“底部抽屉/底部面板”
  - 主区域为预览 + 当前步骤面板
  - 主操作固定在底部（主按钮）
- 平板/桌面：
  - 素材库为左侧栏（固定）
  - 预览在中间，控制面板在右侧

## 4. 页面结构草案

### 4.1 素材步骤（Step 1）
- 顶部：步骤条（素材 / 风格 / 导出）
- 主区：空状态或缩略图网格
- 底部：主按钮“导入素材”
- 抽屉（移动）：素材库缩略图 + 分组

### 4.2 风格步骤（Step 2）
- 主区：预览图 + 一键“推荐风格”
- 下区：风格包卡片横滑
- 次级：批量应用开关（默认开启）
- 进阶入口：
  - “高级调色”按钮（展开二级面板）
  - 预设滤镜保存/管理

### 4.3 导出步骤（Step 3）
- 主区：导出预览（小图）
- 下区：导出配置（格式/质量/尺寸）
- 主按钮：“开始导出”
- 导出完成：提示 + 回到素材库入口

## 5. 关键模块（组件级别）
- WorkspaceShell（步骤条 + 主操作）
- LibraryDrawer / LibraryRail（素材库）
- PreviewCanvas（主预览）
- StylePackCarousel（风格包卡片）
- QuickAdjustPanel（3 主控）
- AdvancedPresetPanel（进阶预设 + 保存）
- ExportSheet（导出设置）
- ProgressToast（导出反馈）

## 6. 交互与状态
- 默认推荐：首次进入风格步骤自动选中推荐风格
- 批量应用：默认为“当前分组全部应用”
- 单张微调：从风格步骤进入并可一键返回
- 恢复进度：再次进入时回到上次步骤与选中风格

## 7. 视觉优先级
- 主按钮始终唯一且高对比
- 预览图为视觉中心，UI 尽量克制
- 风格卡片展示“前后对比”
