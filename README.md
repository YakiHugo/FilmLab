# AI Film Composer

> AI驱动的社媒组图生成系统  
> 像艺术家一样，用统一色调与影调进行摄影表达

---

## ✨ 项目愿景

AI Film Composer 是一个面向创作者的 **AI 驱动社媒组图生成工具**。

它不仅仅是滤镜 App，也不仅仅是修图软件，而是一个：

- 🎞 以“胶片美学”为审美底座
- 🧠 以“AI 理解创作意图”为核心引擎
- 🖼 以“组图叙事表达”为最终输出形式
- 📱 以“社交媒体发布”为使用场景

的创作系统。

一句话总结：

> 像摄影艺术家一样，根据已有作品，用自然语言驱动统一风格创作，并自动生成具有表达力的社媒组图。

---

# 🧠 核心能力架构

系统由四个核心能力模块构成：
自然语言创作层
↓
风格引擎层（Film Look Engine）
↓
图像编辑与生成层
↓
组图编排与输出层

---

# 1️⃣ 自然语言创作引擎

## 目标

将用户的创作意图（自然语言）转化为可执行的视觉风格与结构规则。

## 示例输入

- “冷调、忧郁、东京夜晚、颗粒明显”
- “像宝丽来一样温暖的家庭记忆”
- “90年代港片，强对比，轻微泛绿阴影”

## 输出结构化意图

- 色彩倾向（冷暖、主色偏移）
- 对比曲线特征
- 胶片颗粒强度
- Halation / Bloom 参数
- 构图比例（1:1 / 4:5 / 9:16）
- 是否拼贴 / 是否生成辅助素材

---

# 2️⃣ Film Look Engine（风格核心引擎）

这是产品的灵魂。

## 🎞 胶片仿真能力

- Film Profiles（胶片库）
- Push / Pull
- Print Film 模拟
- Halation（光晕）
- Bloom（高光溢出）
- Film Grain（颗粒）
- Film Compression（动态压缩）
- Dust / Damage / Light Leak（质感层）

## 📷 宝丽来表达支持

- 白色边框模板
- 纸张纹理
- 轻微偏色与低饱和度
- 模拟即时成像色彩响应

## 🎨 风格一致性系统

- 为一组照片生成统一 Look
- 自动适配不同曝光照片
- 保持肤色与关键元素稳定
- 批量处理一致性输出

> 目标：不是“给每张图加滤镜”，而是“为整组图建立一个风格空间”。

---

# 3️⃣ 编辑器底座能力

系统不是单一滤镜工具，而是具备完整编辑能力。

## 基础调整

- 曝光 / 对比 / 高光 / 阴影
- 曲线
- HSL
- 分离色调
- 锐化 / 降噪
- 裁切 / 旋转

## 局部与蒙版

- 主体识别
- 天空识别
- 肤色选择
- 自定义画笔
- 图层与混合模式

## 生成式工具

- Generative Erase（擦除）
- Generative Expand（扩展画布）
- Generative Replace（替换元素）
- AI Upscale（超分辨率）
- 文生图补充资产（贴纸、纹理、氛围图）

---

# 4️⃣ 组图编排系统（核心差异化）

本产品的独特之处。

## 模式一：精选组图

- AI 自动评分与筛选
- 去重与相似图处理
- 生成节奏结构（主图 / 过渡图 / 细节图）

## 模式二：拼贴创作

- 多图自动布局
- 智能裁切与构图平衡
- 版式模板系统
- 宝丽来拼贴风格

## 模式三：混合生成

- 部分图片来自真实摄影
- 部分图片来自文生图
- 统一风格空间融合

---

# 5️⃣ 系统能力要求

## 渲染引擎

- 非破坏性编辑
- GPU 加速
- 参数可回溯
- 高质量色彩管理（LUT 支持）

## 模型架构

- 本地推理 + 云端推理混合
- 风格模型
- 图像理解模型
- 生成式图像模型

## 资产系统

- 胶片库
- 边框模板
- 拼贴模板
- 风格包（可订阅）
